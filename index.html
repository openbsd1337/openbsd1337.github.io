How it Works?
1. Authorized hackspree API clients submit datasourcing requests.
2. Datasourcing requests are voted into a priority queue pipeline.
3. Hackspree Datasourcing Specialists work on most voted datasourcing requests.
4. Hackspree GraphQL Managers integrate the newly sourced data.
